# Privacy Policy — Gopnik from Chertanovo

**App:** Gopnik from Chertanovo (`com.chertanovo.gopnik`)
**Last updated:** 11 July 2026

The mobile game **Gopnik from Chertanovo** ("the App") **does not collect,
transmit, store on any server, or share any personal data.** There are no user
accounts, no login, and no internet connection is required to play.

- **On-device data only.** Game progress (high scores, in-game currency,
  purchased cosmetics, settings) is saved locally using Android's private app
  storage (`SharedPreferences`). It never leaves your device and is removed when
  you uninstall the App.
- **No sensitive permissions.** The App does not access contacts, location,
  camera, microphone, storage, phone state, or any other personal information.
- **No ads, no analytics, no tracking SDKs, no network requests.** As no data is
  collected, none is sold or shared with third parties.
- **Children.** The App knowingly collects no information from anyone, including
  children under 13.

If this policy changes, the updated version will be posted at the same URL with
a new date.
